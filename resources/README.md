# Learning Resources

## Book

The Kubernetes Book - Nigel Poulton (2025 Edition)
https://books.google.co.ke/books?id=enTJEAAAQBAJ

The official documentation tells you what to do. This book explains why. It gave me the mental model to understand the reasoning behind Kubernetes design decisions, which made debugging and building on top of the docs much more effective.
